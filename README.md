# My Python Projects

This repository contains several Python projects that I developed using Visual Studio Code to improve my programming skills.

## Projects

### 1. Calculator with Memory Function (Taschenrechner)

A Python-based calculator with a memory/history function.

Features:
- Addition, subtraction, multiplication and division
- Save calculations in a history file
- Show previous calculations
- Delete calculation history
- Input validation and error handling

Technologies:
- Python
- File handling

---

### 2. Rock Paper Scissors Game (Stein, Papier, Schere)

An interactive Rock-Paper-Scissors game written in Python.

Features:
- User vs computer gameplay
- Random computer choices
- Score tracking
- Loop-based gameplay
- Command line interface

Technologies:
- Python
- Random module



---

### 3. Dice Game (Multiplayer)

A multiplayer dice game written in Python.

Features:
- Multiple players
- Random dice rolls
- Score tracking
- Round-based gameplay
- Automatic winner detection

Technologies:
- Python
- Random module
